데이터 업로드 센터

구글 시트(work.xlsx 대시보드) 구조에 맞춰 정제된 콘텐츠 데이터를 편리하게 등록하세요.

📝 신규 데이터 입력
👁‍🗨 임시 등록 데이터 확인 (미리보기)
ID 카테고리 이모지 제목 요약 태그 날짜 본문 길이
추가된 임시 데이터가 없습니다.
💡 실시간 구글 시트 연동 방법 안내

실제 구글 시트와 웹서버 없이 직결하려면 Google Apps Script (Web App)를 활용하시면 됩니다.

[세팅 방법]
1. 연동하고자 하는 구글 스프레드시트 열기 -> 상단 메뉴 [확장 프로그램] -> [Apps Script] 클릭
2. 아래의 코드를 붙여넣고 저장한 뒤, 우측 상단 [배포] -> [새 배포] (유형: 웹 앱, 액세스 권한: 모든 사용자)로 배포 후 URL을 획득하세요.
function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheets()[0]; var params = JSON.parse(e.postData.contents); // 데이터 구조에 맞춰 추가 [id, category, emoji, title, summary, tags, date, content] sheet.appendRow([ params.id, params.category, params.emoji, params.title, params.summary, params.tags, params.date, params.content ]); return ContentService.createTextOutput(JSON.stringify({"result":"success"})) .setMimeType(ContentService.MimeType.JSON); }
